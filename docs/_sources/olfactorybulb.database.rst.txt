olfactorybulb.database
=========================================

The following classes are `Object-Relation-Map (ORM) <https://en.wikipedia.org/wiki/Object-relational_mapping>`_
objects that directly map to tables in the SQLite database.

Class properties map directly to columns of each table.
These classes are used to query the model database using Python with the `peewee <http://docs.peewee-orm.com/en/latest/>`_
package.

The database can be found under::

    [repo]/olfactorybulb/model-data.sqlite
